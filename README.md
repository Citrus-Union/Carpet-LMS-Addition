# Carpet LMS Addition

Carpet LMS Addition is a small Carpet extension, still in progress.

English | [中文](README_cn.md)

## Rules

### pearlIgnoreEntityCollision

Ender pearls ignore entity collisions (no teleport, fly through like air).

Categories: `lms`, `survival`.

Default: `false`.

### playerCommandDropall

Permission gate for /player <name> dropall (true/false/ops/0-4).

Categories: `lms`, `survival`.

Default: `false`.

### fragileVaults

Vaults match beacon blast resistance (TNT can break them).

Categories: `lms`, `survival`.

Default: `false`.

### fragileTrialSpawners

Trial spawners match normal spawner blast resistance.

Categories: `lms`, `survival`.

Default: `false`.

### allayHealInterval

Ticks between Allay passive heals (vanilla 10; minimum 1).

Categories: `lms`, `survival`.

Default: `10`.

### commandPlace

Permission gate for /place (uses Carpet permissions: `true`, `false`, `ops`, `0-4`).

Categories: `lms`, `survival`.

Default: `ops`.
